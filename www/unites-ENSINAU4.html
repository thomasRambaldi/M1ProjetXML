Unite : Programmation fonctionnelle

Unité : Programmation fonctionnelle

id : ENSINAU4
unité :
Credits : 3
Résume :
1- Introduction à la programmation fonctionnelle, sur l'exemple du langage OCaml Expressions, évaluation, types de base. Polymorphisme, ordre supérieur. Fonctions récursives, filtrage. Les diverses stratégies d'évaluation, application à la programmation d'un opérateur de point fixe. 2- Les types Sommes, types récursifs, polymorphes. Arbres. Filtrage. 3- Sémantique opérationnelle Liaisons, environnements, clôtures. Évaluation des fonctions récursives . 4- Aspects impératifs Exceptions, entrées-sorties, séquencement, fichiers, références, tableaux, enregistrements. 5- Implantation du filtrage en OCaml Termes formels, substitutions, filtrage.
Plan : Cours/TD/TP : 10h/10h/10h
