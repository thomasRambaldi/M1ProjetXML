Unite : Web Multimédia

Unité : Web Multimédia

id : ENSINCU62
unité :
Credits : 3
Résume :
Cette UE analyse les problèmes fondamentaux et techniques que pose le développement d'applications multimédia (images, vidéo, son, etc.) et l’architecture logicielle nécessaire pour supporter le traitement de telles données. Cela permet d'abord une familiarisation avec les problèmes de streaming, de VoD, etc. Seront ensuite étudiés les problèmes que pose la mise en ligne de telles applications : passage à l’échelle, tolérance aux pannes, etc. Enfin, nous étudions l’intégration des services de téléphonie sur IP dans une telle application tout en considérant la personnalisation d’accès à l’information multimédia et aux services disponibles.
Plan : Cours/TD/TP : h/h/h
Lieu :

Cette unité appartient au parcours :
  • M2 Spécialité Systèmes d'information répartis (SIR)
