Unite : Sémantique

Unité : Sémantique

id : ENSINBU8
unité : option
Credits : 3
Résume :
1-Le lambda-calcul simplement typé Syntaxe, typage, beta et eta-conversions, équations. Modèles : interprétation des jugements de typage. 2-Extension à PCF. La règle \mu. 3-Points fixes dans les ordres partiels complets. Le théorème de Tarski. 4-Sémantique par point fixe de PCF. 5-Cours de clôture : survol des différentes approches sémantiques. Sémantiques opérationnelles : par transitions, naturelles, machine abstraite de Landin. Sémantique dénotationnelles : cas fonctionnel(cf.chapitre 4) et cas d'un petit langage impératif.
Plan : Cours/TD/TP : 10h/10h/10h
Lieu : Luminy
